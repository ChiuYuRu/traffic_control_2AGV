# traffic_control_2AGV

#### 環境依賴

VS code v1.57.1
C/C++ms-vscode.cpptools v1.5.0-insiders

#### 部署步驟
1. 以matlab(`get_map.m`)轉換地圖得到`地圖.txt`(此範例為`clear_map.txt`)
2. 當前資料夾開啟terminal
3. `$ make` 以產生traffic_control_2AGV_enter.out
4. `$./traffic_control_2AGV_enter.out`
5. 即有執行檔
